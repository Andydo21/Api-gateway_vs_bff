# Microservices Architecture: Pitch-Booking & Startup Ecosystem

Một hệ thống quản lý hệ sinh thái startup và đặt chỗ thuyết trình (pitching) hiện đại, sử dụng kiến trúc **Microservices**, **BFF (Backend for Frontend)**, và **Apache APISIX Gateway**.

---

## 🚀 Quick Start

### 1. Khởi động toàn bộ hạ tầng (Docker)
```bat
# Tự động hóa hoàn toàn trên Windows
start.bat
```

# Tạo superuser
python manage.py createsuperuser

# Chạy development server
python manage.py runserver

# Chạy tests
python manage.py test
```
